# Git-Helper - Cli
=============
Git-Helper is a program that allows users to simplify the experience with git.
It is crossplatform *, so it works with all OS, as far as Python, with all the modules, and git are installed.

# Installation
If you are running the program for the first time, make sure to execute requirement.py, so that you have all the modules.
If you want to know the requirements are:

- Python => 3.0
- Git
- gitpython (module)
- Pathlib (module) **
- OS (module) **
- Platform (module) **
- Shutil (module)
- Stat (module)

**These module should come preinstalled with Python

# How to Use
For now the only way to run the program is by executing "Git HelperCli.py".
