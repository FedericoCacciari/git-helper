# git-helper
Git-helper
Programma in Python che permette di semplificare l'utilizzo delle funzioni basiche di git.
Il programma può essere importato come modulo importando il file "__init__.py"
come programma asestante con il file "main.py"
o da linea di comando con argomenti.

Funzionalità già inserite:
- Clone di Repository, sostituzione dei file nella cartella

Da inserire:

- Argomenti da linea di comando
- Push
